Guia basica sobre el uso de esta aplicacion de gestion

Versión de la aplicación: 1.0

En esta pantalla de ayuda se se explicara con detalle como se usa esta aplicacion para gestionar el inventario del instituto.

Tipos de usuario de la aplicacion

En la aplicación podremos encontrar dos tipos diferentes de usuario:

  • Usuario Normal
  • Administrador

La principal diferencia que existe entre estos dos tipos de usuarios es que el administrador posee funciones que el usuario normal no podría realizar
