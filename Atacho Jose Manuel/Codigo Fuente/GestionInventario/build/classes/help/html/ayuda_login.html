Pantalla de login

Esta es la pantalla que aparece cada vez que se arranque el programa, en ella se puede apreciar el logo del instituto, un campo para introducir el email del usuario y la clave de este. Si el usuario no existe o la contraseña no es la correcta, el programa lanzara un mensaje al usuario

Imagen de la ventana de inicio

Al hacer clic en el enlace de la esquina inferior derecha, este nos abrira una ventana en la cual podremos modificar los parametros de conexion a nuestra base de datos, los datos de conexion por defecto son los siguientes

  • Ip: localhost
  • Usuario: root
  • Contraseña: root
