Menu Principal

Esta es la pantalla que recibira al usuario tras haber iniciado sesion, lo primero que apreciara el usuario es una tabla en la cual se recoje todos los materiales actualmente registrados en la base de datos.
En la parte superior están las diferentes funciones que posee el programa además de poder ver la sección de ayuda, si el usuario es administrador este tendra disponible dos opciones extra:

  • Gestion de usuario
  • Otros
